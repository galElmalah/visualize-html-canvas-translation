How translation work

Basically it moves the origin to a new x,y coordinates.

In this example - Each time we translate the coordinates system to a new point and draw in 0,0.

If we bundle a couple of translations calls together we will get a translation that is the sum of all of those translations.
For example:
ctx.translate(40,40);
ctx.translate(40,40);
Is equivlent to: ctx.translate(80,80);.

This property, where transformation can be added together applies to all canvas transformations

*This process is done by using the translation matrix beahind the seen as seen below
*Using save and restore between each call
ctx.save();
ctx.translate(tx,ty);
drawPoint(0,0);
ctx.restore();

0 1 ty
